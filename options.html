Anthropic API Key

  1. Create a free account at console.anthropic.com.
  2. Add credit under Billing (classification uses the low-cost Claude Haiku model).
  3. Open API Keys and click Create Key.
  4. Copy the key (it starts with sk-ant-) and paste it below.
Stored locally in your browser. Never sent anywhere except api.anthropic.com.

Appearance

Applies to the popup and this settings page.

Privacy & data

When you run an analysis, the domain names you analyze are sent to the Anthropic API (using your key) and to DuckDuckGo for short descriptions. To improve results, the extension also fetches the public homepage of each newly-analyzed domain directly — so your browser connects to those domains from your IP address. Only the page title and meta description are read; no cookies or credentials are sent, and results are cached so each domain is contacted at most once.

Full details: Privacy Policy.

Classification Cache

Classifications are cached locally per site for 90 days so they aren't re-sent to the AI; after that they're automatically re-checked. Caching per site means a domain can be classified differently depending on the page it appears on. Clear the cache to force fresh classifications.
